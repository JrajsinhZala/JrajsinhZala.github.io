# Jitendrasinh Zala - Portfolio Website

A modern, responsive portfolio website showcasing my work as a Senior React Native Developer with 8+ years of experience.

## Features

- Modern dark theme with vibrant green accents
- Animated typing effect in hero section
- Custom cursor effect (desktop)
- Fully responsive layout with hamburger menu
- Smooth scrolling navigation with animations
- Professional skills section with progress bars
- Projects showcase with hover effects
- Contact form and social media integration
- Optimized for React Native/Mobile Development portfolio

## Getting Started

1. Open `index.html` in your web browser to view the website
2. The portfolio is already customized with Jitendrasinh Zala's information
3. To further customize:
   - Update projects in the Projects section
   - Add your profile photo to `assets/images/profile.jpg`
   - Add project screenshots to `assets/images/project1.jpg`, `project2.jpg`, `project3.jpg`
   - Modify skills percentages if needed
4. Customize colors in `assets/css/styles.css` CSS variables section

## Customization

### Colors
Edit the CSS variables in `styles.css` to change the color scheme:
```css
:root {
    --primary-color: #00ff88;        /* Vibrant green */
    --secondary-color: #00d9ff;      /* Cyan accent */
    --bg-primary: #0a192f;           /* Dark navy */
    --bg-secondary: #112240;         /* Lighter navy */
    --text-primary: #ccd6f6;         /* Light text */
    --text-secondary: #8892b0;       /* Muted text */
}
```

### Fonts
Change the font family in the `body` selector in `styles.css`.
